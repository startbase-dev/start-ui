# Amount Input

The AmountInput component is a customizable input field designed for users to enter numerical amounts, featuring validation and formatting options.

## Installation

<Tabs items={["npm", "yarn", "pnpm"]}>
<Tabs.Tab>
    ``` bash copy
    npm install --save @start-base/react-form-elements
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy
    yarn add @start-base/react-form-elements
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy
    pnpm add @start-base/react-form-elements
    ```
  </Tabs.Tab>
</Tabs>

Make sure to add css file to your app root file

``` tsx copy
import '@start-base/react-form-elements/styles.css';
```
