# OTPInput

The OTPInput component is designed for secure and user-friendly input of One-Time Passwords (OTPs). It features multiple input fields to capture each digit of the OTP individually, providing a clear and intuitive interface for users.

## Installation

<Tabs items={["npm", "yarn", "pnpm"]}>
<Tabs.Tab>
    ``` bash copy
    npm install --save @start-base/react-form-elements
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy
    yarn add @start-base/react-form-elements
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy
    pnpm add @start-base/react-form-elements
    ```
  </Tabs.Tab>
</Tabs>

Make sure to add css file to your app root file

``` tsx copy
import '@start-base/react-form-elements/styles.css';
```
