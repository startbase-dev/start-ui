# NumberInput

The NumberInput component is designed for precise and efficient numeric input. It features a clean and intuitive interface, allowing users to enter numbers directly or adjust values using increment and decrement buttons.

## Installation

<Tabs items={["npm", "yarn", "pnpm"]}>
<Tabs.Tab>
    ``` bash copy
    npm install --save @start-base/react-form-elements
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy
    yarn add @start-base/react-form-elements
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy
    pnpm add @start-base/react-form-elements
    ```
  </Tabs.Tab>
</Tabs>

Make sure to add css file to your app root file

``` tsx copy
import '@start-base/react-form-elements/styles.css';
```
