# Countdown

A customizable React countdown component that dynamically displays only active time units, supporting multiple formats and precise millisecond accuracy.

<ApiReferenceButton path="form-input--docs" />

## Installation

<Steps>

### Install the package

<Tabs items={["npm", "yarn", "pnpm"]}>
<Tabs.Tab>
    ``` bash copy
    npm install --save @start-base/start-ui
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy
    yarn add@start-base/start-ui
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy
    pnpm add @start-base/start-ui
    ```
  </Tabs.Tab>
</Tabs>
 
### Add the styles
 
Make sure to add css file to your app root file

``` tsx copy
import '@start-base/start-ui/styles.css';
```
