# Countdown

A customizable React countdown component that dynamically
displays only active time units, supporting multiple
formats and precise millisecond accuracy.

## Installation

<Steps>

### Install the package

```sh copy npm2yarn
npm i @start-base/start-ui
```

### Add the styles

<Tabs items={["Next.js (App Router)", "Next.js (Pages Router)", "React.js"]}>
    <Tabs.Tab label="Next.js (App Router)">
        To use the React form elements styles in a Next.js project, import the styles in your `layout.tsx` file.

        ```tsx copy filename="app/layout.tsx"
        import "@start-base/start-ui/styles.css";
        ```

    </Tabs.Tab>
    <Tabs.Tab label="Next.js (Pages Router)">
        For Next.js (Pages Router) projects, ensure you include the React form elements styles in the `_app.tsx` file.

        ```tsx copy filename="pages/_app.tsx"
        import @start-base/start-ui/styles.css";
        ```

    </Tabs.Tab>
    <Tabs.Tab label="React.js">
        In a React.js project, you should import the styles directly into your `App.js` file.

        ```tsx copy filename="App.js"
        import @start-base/start-ui/styles.css";
        ```

    </Tabs.Tab>

</Tabs>

</Steps>

## API

| Name           | Type              | Default   | Description                                                            |
| -------------- | ----------------- | --------- | ---------------------------------------------------------------------- |
| date           | `Date` - `number` |           | Countdown Date                                                         |
| showDay        | `bolean`          | true      | Show day                                                               |
| showHour       | `bolean`          | true      | Show hour                                                              |
| showMinute     | `bolean`          | true      | Show minute                                                            |
| showSecond     | `bolean`          | true      | Show second                                                            |
| type           | `string`          | `default` | `withSeparator` - `withLabels` - `labelsUnder` - `inBoxes` - `default` |
| itemClassName  | `string`          |           | Item className                                                         |
| rootClassName  | `string`          |           | Root className                                                         |
| valueClassName | `string`          |           | Value className                                                        |
| labelClassName | `string`          |           | Label className                                                        |
