# Toggle

The Toggle component is a switch button that allows users to toggle between two states, such as on/off or active/inactive. It supports various configurations for color, size, variant, and fluidity.

import Preview from "../../../components/Preview/Preview";
import { Toggle } from "@start-base/start-ui";
