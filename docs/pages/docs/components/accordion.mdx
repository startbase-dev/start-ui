# Accordion

A series of interactive headings arranged vertically, each revealing a section of content when clicked.

## Installation

<Steps>
### Install the package
```sh copy npm2yarn
npm i @start-base/start-ui
```
### Add the styles

<Tabs items={["Next.js (App Router)", "Next.js (Pages Router)", "React.js"]}>
    <Tabs.Tab label="Next.js (App Router)">
        To use the React form elements styles in a Next.js project, import the styles in your `layout.tsx` file.

        ```tsx copy filename="app/layout.tsx"
        import "@start-base/start-ui/styles.css";
        ```

    </Tabs.Tab>
    <Tabs.Tab label="Next.js (Pages Router)">
        For Next.js (Pages Router) projects, ensure you include the React form elements styles in the `_app.tsx` file.

        ```tsx copy filename="pages/_app.tsx"
        import "@start-base/start-ui/styles.css";
        ```

    </Tabs.Tab>
    <Tabs.Tab label="React.js">
        In a React.js project, you should import the styles directly into your `App.js` file.

        ```tsx copy filename="App.js"
        import "@start-base/start-ui/styles.css";
        ```

    </Tabs.Tab>

</Tabs>

### Import the component

    <Tabs items={["Individual", "Global"]}>
        <Tabs.Tab label="Individual">

            ```tsx copy
            import Accordion from '@start-base/start-ui/Accordion';
            ```

        </Tabs.Tab>
        <Tabs.Tab label="Global">

            ```tsx copy
            import { Accordion } from '@start-base/start-ui';
            ```

        </Tabs.Tab>

    </Tabs>

</Steps>

## API

| Name               | Type      | Default    | Description                       |
| ------------------ | --------- | ---------- | --------------------------------- |
| summary            | `string`  |            | Summary of the accordion          |
| children           | `string`  |            | Children of the accordion         |
| actions            | `object`  |            | Actions of the accordion          |
| className          | `string`  |            | Classname the accordion           |
| actionsClassName   | `string`  |            | Action Classname the accordion    |
| summaryClassName   | `string`  |            | Summary Classname the accordion   |
| containerClassName | `string`  |            | Container Classname the accordion |
| defaultExpanded    | `boolean` | `false`    | Default Expanded the accordion    |
| disabled           | `boolean` | `false`    | Disabled the accordion            |
| unmountOnExit      | `boolean` | `false`    | Unmount the accordion on exit     |
| variant            | `string`  | `"drawer"` | Styling variant of the accordion  |
