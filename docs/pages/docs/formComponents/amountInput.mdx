# Amount Input

The AmountInput component is a customizable input field designed for users to enter numerical amounts, featuring validation and formatting options.

<ApiReferenceButton path="form-amountinput--docs" />

## Installation

<Steps>

### Install the package

<Tabs items={["npm", "yarn", "pnpm"]}>
  <Tabs.Tab>
    ``` bash copy npm install --save @start-base/react-form-elements ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy yarn add @start-base/react-form-elements ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy pnpm add @start-base/react-form-elements ```
  </Tabs.Tab>
</Tabs>
### Add the styles Make sure to add css file to your app root file

```tsx copy
import "@start-base/react-form-elements/styles.css";
```

</Steps>
