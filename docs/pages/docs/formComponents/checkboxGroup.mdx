# CheckboxGroup

The Checkbox Group component provides a user-friendly interface for toggling between checked and unchecked states, with customizable labels and state management.

<ApiReferenceButton path="form-checkboxGroup--docs" />

## Installation

<Steps>

### Install the package

<Tabs items={["npm", "yarn", "pnpm"]}>
<Tabs.Tab>
    ``` bash copy
    npm install --save @start-base/react-form-elements
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy
    yarn add @start-base/react-form-elements
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy
    pnpm add @start-base/react-form-elements
    ```
  </Tabs.Tab>
</Tabs>
 
### Add the styles
 
Make sure to add css file to your app root file

``` tsx copy

```
