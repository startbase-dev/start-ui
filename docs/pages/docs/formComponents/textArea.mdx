# TextArea

The TextArea component is designed for multi-line text input, providing users with an area to enter larger amounts of text. It features a flexible and resizable interface that adapts to user input

<ApiReferenceButton path="form-textArea--docs" />

## Installation

<Steps>

### Install the package

<Tabs items={["npm", "yarn", "pnpm"]}>
  <Tabs.Tab>
    ``` bash copy npm install --save @start-base/react-form-elements ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy yarn add @start-base/react-form-elements ```
  </Tabs.Tab>
  <Tabs.Tab>
    ``` bash copy pnpm add @start-base/react-form-elements ```
  </Tabs.Tab>
</Tabs>
### Add the styles Make sure to add css file to your app root file

```tsx copy
import "@start-base/react-form-elements/styles.css";
```
